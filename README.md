# Royal Retreat — SPA Hotel

**Royal Retreat** — a modern SPA hotel website with online booking for rooms and cottages. The project includes a React frontend and a Node.js backend with PostgreSQL database.

## 🔗 Demo

https://royal-retreat-en.vercel.app

https://github.com/KatrinCl/RoyalRetreat-en

---

## 🏨 About the Project

Royal Retreat hotel is a city-located place for pleasant and comfortable leisure within Russia's "Golden Ring". The project provides guests with the ability to:

- Browse available rooms, cottages, and apartments
- Book accommodation for selected dates
- Explore spa complex services, banquet halls, and infrastructure
- Contact the hotel administration

---

## ✨ Features

### For Guests:
- 🏠 Browse available rooms and cottages
- 📅 Search by check-in/check-out dates and number of guests
- 💰 Automatic accommodation cost calculation
- 🛒 Book rooms and cottages
- 👤 User registration and authentication
- 📝 Personal dashboard with booking history

### Admin Panel:
- ➕ Add/edit rooms and cottages
- 🖼️ Image upload (Cloudinary)
- 📊 Booking management
- 👥 User management
